## Требования

* python >= 3.8
* poetry >= 1.1.4
* make >= 4

*Образец описания команд, необязательно именно так*

## Установка и запуск

* `make install` установка утилиты
* `gendiff -h` запуск утилиты (вызов справки)

## Дополнительно

* `make lint` проверка линтером
* `make test` проверка тестами
